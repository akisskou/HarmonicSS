Instruction for developers


Request execution POST service

Service url:

http://harm.harmonicss.eu:8989/HarmonicSS/PatientSelectionImpl

json input example, for user test1 to execute request with darID=349:

{"username":"test1","password":"1test12!","requestID":"349"}

json output, xml response string:

{"responseXML":"..."}


View execution data GET service

Service url, with appropriate parameters to view execution data of request with darID=350, executed by user test1:

http://harm.harmonicss.eu:8989/HarmonicSS/ReqRespList?userID=test1&requestID=350

json output, execution data list, with data of each execution as json object:

{"request_response_list":[{...},...,{...}]}


View patient IDs POST service

Service url:

http://harm.harmonicss.eu:8989/HarmonicSS/FindIDsByRequestID

json input example, for user test2 to view patient IDs execution results of request with darID=350, executed at cohort 35:

{"username":"test2","password":"1test22!","requestID":"350","mycohort":"35"}

json output, a list of patient IDs lists, with data of each execution as json object:

{"patient_IDs_list":[{...},...,{...}]}
